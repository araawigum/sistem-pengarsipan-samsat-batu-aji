# Sistem Pengarsipan Data Wajib Pajak Samsat Batu Aji Berbasis Web

## Deskripsi

Sistem Pengarsipan Data Wajib Pajak Samsat Batu Aji Berbasis Web merupakan aplikasi yang dikembangkan untuk membantu proses pengelolaan arsip data wajib pajak kendaraan bermotor secara lebih efektif, terstruktur, dan terdokumentasi. Sistem ini memudahkan proses penyimpanan, pencarian, pengelolaan, serta pembuatan laporan data wajib pajak.

## Fitur

- Login pengguna
- Dashboard
- Manajemen Data Wajib Pajak
- Manajemen Data Kendaraan
- Manajemen Data Pembayaran
- Manajemen Arsip Dokumen
- Manajemen Data User
- Pencarian data
- Cetak laporan

## Teknologi

- PHP Native
- MySQL / MariaDB
- HTML
- CSS
- JavaScript
- XAMPP

## Struktur Folder

```
assets/
config/
pages/
index.php
login.php
logout.php
db_arsip_samsat.sql
```

## Cara Menjalankan

1. Install XAMPP.
2. Aktifkan Apache dan MySQL.
3. Import database `db_arsip_samsat.sql` ke phpMyAdmin.
4. Simpan project pada folder `htdocs`.
5. Buka browser dan akses:

```
http://localhost/arsip-samsat
```
